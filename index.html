      🚩 Currently, I'm building a School Management System for a client

      29-04-2026 - Present

      What I'm building:

      • ✓ Architecting and building a full-stack School Management System end-to-end for a client — covering both a Next.js 15 frontend and a Node.js/Express REST API backend.
      • ✓ Role-based dashboards for five roles (superadmin, admin, teacher, student, parent) enforced across both the frontend and backend via a dedicated RBAC layer.
      • ✓ Secure JWT authentication with httpOnly cookies, refresh token rotation, and role-based route protection handled server-side in both Next.js middleware and Express guards.
      • ✓ A PostgreSQL database with a carefully designed schema using Prisma ORM — modelling complex academic relationships across users, roles, classes, grades, and more.
      • ✓ Backend features including file uploads, email notifications, and a reporting & analytics module surfaced through a clean REST API consumed by TanStack Query on the frontend.
      • ✓ Bilingual English/Arabic interface with full RTL support via next-intl.
      • ✓ A layered frontend architecture (apiClient → API layer → query hooks → components) with TanStack Query and Zustand, and type-safe forms via React Hook Form and Zod.
      • ✓ Currently Working on the Dashboard Page.
